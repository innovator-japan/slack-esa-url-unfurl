## slack-esa-url-unfurl
- Slackに貼られたesaのURLを展開する
- https://github.com/FromAtom/Kujaku のAWS Lambda(Node.js)版実装
- 必要な環境変数
    - SLACK_OAUTH_ACCESS_TOKEN
    - ESA_TEAM_NAME
    - ESA_ACCESS_TOKEN
